Tic Tac Toe Game (Java)

📌 Project Description

This project is a simple Tic Tac Toe game developed using Java.
The game allows a human player to play against a computer (AI) player in the console.
The board is a 3 × 3 grid, and the players take turns placing their marks (X and O) until one player wins or the game ends in a draw.

🚀 Features

* Console-based Tic Tac Toe game
* Human vs Computer gameplay
* Random AI move generation
* Win detection (Row, Column, Diagonal)
* Draw detection
* Input validation for moves

🛠 Technologies Used

* Java
* Object Oriented Programming (OOP)
* Eclipse IDE

📂 Project Structure

* LaunchGame – Initializes and displays the game board
* Player – Abstract class for players
* HumanPlayer – Handles human player input
* AIPlayer – Generates random moves for the computer
* Tictactoe – Main class to run the game

▶️ How to Run the Program

1. Clone the repository
2. Open the project in Eclipse or any Java IDE
3. Run the Tictactoe.java file
4. Enter row and column values (0–2) to place your mark

Example input:
0 1

🎯 Game Rules

* Player X starts the game
* Players take turns placing marks
* First player to align 3 marks in a row, column, or diagonal wins
* If all cells are filled without a winner, the game is a draw
